## 1. 環境構築

```bat
:: リポジトリのルートディレクトリで実行してください
:: Run from the repository root directory

:: 仮想環境の作成（初回のみ）
python -m venv venv

:: 仮想環境の有効化
venv\Scripts\activate

:: 依存関係インストール（初回 or 要件変更時）
pip install -r requirements.txt

:: アプリ起動
python main.py
```

## 4. テスト時の確認ポイント

- **ログファイル**: エラーがある場合は `%USERPROFILE%\.csview\csview.log` を確認
- **一時ファイル**: テスト後に `%TEMP%\csv_editor_*.db` が残っていないか確認（タブを閉じると自動削除される）
- **状態ファイル**: `%USERPROFILE%\.csview\state.json` にバージョン情報が書き込まれているか確認

## 5. ZIPファイルを使うテスト（T35）の準備手順

T35 のテストには、T26 の分割保存で作成したZIPが必要です。
以下の順で実施してください：

1. `test_05_large.csv` を開く
2. 「ファイル → 名前を付けて分割して保存」
3. 分割数: `5`、ファイル名: `test_split` で保存
4. 保存されたZIPを「ファイル → csvを結合して開く」で開く
5. 1000行のデータが結合されて表示されることを確認
